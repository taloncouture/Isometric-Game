# About
This is based on the **New Beginner Java Game Programming Tutorial Series** by [Codenmore](https://github.com/CodeNMore/New-Beginner-Java-Game-Programming-Src)

This is obviously very outdated \(12 years at the time of writing this!\) But this is mostly just go get a little more comfortable with Java, which I
have not used in quite a while as well as to better understand game engines and rendering logic \(especially since isometric rendering gets weird\)

So this probably will not become a real game, but it might still be playable, but at the very least will be a fun side project!

# Code
This is all in vanilla Java without any libraries, just using JavaFX.
